¿Qué es un Sistema Operativo?

Es un programa o software que controla la ejecución de aplicaciones y programas, actúa como interfaz entre las aplicaciones y el hardware de la computadora.

Tipos de Sistemas Operativos

De mainframe

Sistemas operativos de mainframe

Los sistemas operativos para las mainframes están profundamente orientados hacia el procesamiento de muchos trabajos a la vez, de los cuales la mayor parte requiere muchas operaciones de E/S.

Por lo general estos sistemas operativos ofrecen tres tipos de servicios: Procesamiento por lotes: sistema de reclamos en una aseguradora. Procesamiento de transacciones: sistema de reservas en una aerolínea. Tiempo compartido: entornos donde varias personas necesitan utilizar el sistema al mismo tiempo, como en empresas

De servidores

De servidores

Los sistemas operativos de servidores se ejecutan, como su nombre lo indica, en servidores los cuales proporcionan múltiples servicios a diferentes máquinas a través de una red, también son los encargados de la gestión de los recursos de software y del hardware, y ofrecen capacidades para ejecutar, administrar, monitorear y controlar aplicaciones, procesos y dispositivos cliente.

De multiprocesadores

Sistemas operativos de multiprocesadores

En un sistema de un solo procesador, los hilos se ejecutan uno tras otro de forma temporal. Esto contrasta con un sistema multiprocesador, donde varios hilos se ejecutan al mismo tiempo, uno en cada procesador disponible.

Estos sistemas necesitan sistemas operativos especiales, que comúnmente son variaciones de los sistemas operativos de servidores con características especiales para la comunicación, conectividad y consistencia.

Muchos sistemas operativos populares (incluyendo Windows y Linux) se ejecutan en multiprocesadores.

Integrados

Sistemas operativos integrados

Operan en dispositivos que no se consideran generalmente como computadoras, ya que no aceptan software instalado por el usuario.

Ejemplos más comunes:Microondas, televisores samart, autos.

De computadoras y móviles personales

Sistemas operativos de computadoras y móviles personales

Estos sistemas operativos actúan como un intermediario entre el hardware y el usuario, gestionando los recursos del sistema y permitiendo la ejecución de aplicaciones ya sea en computadoras, como móviles personales.

Sistemas operativos de computadoras personales: Windows, MacOs, Linux.

Sistemas operativos de móviles personales: Android, iOS.

De nodos sensores

Sistemas operativos de nodos sensores

Los nodos sensores contienen un sistema operativo que por lo general es un manejador de eventos, que responden a cambios externos. Además estos ya tienen un sistema integrado, es decir que un usuario no descarga programas para este mismo.Estos sistemas se usan para detectar incendios, medir las temperaturas, y las precipitaciones. Ejemplo: TinyOS, Contiki, RIOT OS, LiteOS, Nano-RK.

De tarjetas inteligentes

Sistemas operativos de tarjetas inteligentes

Una tarjeta inteligente, normalmente un tipo de tarjeta con chip, es una tarjeta de plástico que contiene un chip informático integrado que almacena y procesa datos. Tienen severas restricciones de poder de procesamiento y memoria.

El sistema operativo de la tarjeta es el software que se programa en el microprocesador de la tarjeta inteligente, el cual le permite a las tarjetas: Almacenar datos. Encriptar información. Leer y escribir datos. Restringir el acceso a los datos personales del usuario y negocio, mediante un mecanismo de control.

En tiempo real

Sistemas operativos en tiempo real

Los sistemas operativos de tiempo real son aquellos que ejecutan aplicaciones a tiempo real cuando se necesita un respuesta inmediata, normalmente este tiempo de respuesta se mide entre milisegundo o hasta microsegundos.

Estos sistemas se caracterizan por su programación preventiva

De Computadora de bolsillo

Sistemas operativos de computadora de bolsillo

Los sistemas operativos que operan en estos dispositivos de bolsillo son cada vez más sofisticados, con la habilidad de proporcionar telefonía, fotografía digital y otras funciones. Muchos de ellos también ejecutan aplicaciones desarrolladas por terceros Dos de los sistemas operativos más populares para los dispositivos de bolsillo son Android e iOS, el último PDA tiene un sistema operativo Windows 10 acompañado de 4 GB de memoria RAM y de 64 GB de almacenamiento interno.
