Tipos de sistemas operativos

Aquí se detallara los tipos de sistemas operativos

De mainframe

Sistemas operativos de mainframe

Los sistemas operativos para las mainframes están profundamente orientados hacia el procesamiento de muchos trabajos a la vez, de los cuales la mayor parte requiere muchas operaciones de E/S. Por lo general estos sistemas operativos ofrecen tres tipos de servicios: Procesamiento por lotes Procesamiento de transacciones Tiempo compartido

De servidores

De servidores

Los sistemas operativos de servidores se ejecutan, como su nombre lo indica, en servidores los cuales proporcionan múltiples servicios a diferentes máquinas a través de una red, también son los encargados de la gestión de los recursos de software y del hardware, y ofrecen capacidades para ejecutar, administrar, monitorear y controlar aplicaciones, procesos y dispositivos cliente.

De multiprocesadores

Sistemas operativos de multiprocesadores

Antes de definir a los sistemas operativos es importante saber qué son los sistemas multiprocesadores. Para obtener mayor cómputo, en la actualidad, es común conectar varias CPU en un solo sistema. Dependiendo de la exactitud con la que son conectadas y de lo que se comparta estos sistemas como multiprocesadores.En un sistema de un solo procesador, los hilos se ejecutan uno tras otro de forma temporal. Esto contrasta con un sistema multiprocesador, donde varios hilos se ejecutan al mismo tiempo, uno en cada procesador disponible. Estos sistemas necesitan sistemas operativos especiales, que comúnmente son variaciones de los sistemas operativos de servidores con características especiales para la comunicación, conectividad y consistencia. Muchos sistemas operativos populares (incluyendo Windows y Linux) se ejecutan en multiprocesadores.

Integrados

Sistemas operativos integrados

Operan en las computadoras que controlan dispositivos que no se consideran generalmente como computadoras, ya que no aceptan software instalado por el usuario.

Ejemplos más comunes:Microondas, televisores samart, autos.

De computadoras y móviles personales

Sistemas operatvios de computadoras y móviles personales

Estos sistemas operativos actúan como un intermediario entre el hardware y el usuario, gestionando los recursos del sistema y permitiendo la ejecución de aplicaciones ya sea en computadoras, como móviles personales.

Sistemas operativos de computadoras personales: Windows, MacOs, Linux.

Sistemas operativos de móviles personales: Android, iOS.

De nodos sensores

Sistemas operativos de nodos sensores

Los nodos sensores contienen un sistema operativo que por lo general es un manejador de eventos, que responden a cambios externos. Además estos ya tienen un sistema integrado, es decir que un usuario no descarga programas para este mismo.

De tarjetas inteligentes

Sistemas operativos de tarjetas inteligentes

Estas son un dispositivo de seguridad que ofrece funciones de almacenamiento seguro de datos y también el procesamiento de la misma. Estas tarjetas tienen un chip empotrado que hace que se implemente un sistema de ficheros cifrados y funciones criptográficas. Para poder acceder a la información de esta se necesita un lector para el chip.

Los sistemas operativos de tiempo real son aquellos que ejecutan aplicaciones a tiempo real cuando se necesita un respuesta inmediata, normalmente este tiempo de respuesta se mide entre milisegundo o hasta microsegundos.

Estos sistemas se caracterizan por su comportamiento determinista, programación preventiva y herencia prioritaria
